The syntax to set a trap in a room is as follows:
: *set r x <Trap#>

To set the exit room for a trap (or for room flags 1 and 63)
: *set r xx <room#>

To set the trigger weight for a trap:
: *set r xw <#>

To set the strength of a trap:
: *set r xs <#>       used by piercers (number of piecers)
                      and traps that poison (strength of poison)
