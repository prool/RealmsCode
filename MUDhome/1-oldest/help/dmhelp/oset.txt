	*set o <name> [<crt>] <ad|ar|dn|ds|dp|f|m|sm|s|v|wg|wr|key> [<val>]

	ad	adjustment
	ar	armor
	dn	number of dice
	ds	sides of dice
	dp	bonus/plus
	f#	flag toggle
	m	magic power
	s	shots current
	sm	shots max
	v	value
	wg	weight
	wr	wear location flag
	key	key number

You can also set an effect on an object - this effect at the specified strength
will be applied to the player when they eat/drink the item.

       *set o <object> effect <effect> <duration> <strength>
